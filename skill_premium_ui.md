# Skill: Refinamiento Estético Moderno (21st.dev)

## Definición
Esta skill permite elevar la interfaz de aplicaciones legales a un estándar de élite, utilizando técnicas modernas de diseño UI/UX sin comprometer la integridad de los datos.

## Principios de Diseño
1. **Glassmorphism**: Uso extensivo de `backdrop-filter: blur()` y fondos con opacidad controlada para crear profundidad.
2. **Bento Grids**: Estructura de rejilla modular y asimétrica para organizar contenidos variados (Holdings, Podcasts).
3. **Micro-animaciones**: Transiciones suaves y efectos de hover que dan vida a la interfaz.
4. **Shimmer Borders**: Efectos de brillo perimetral en tarjetas y botones destacados.
5. **Paleta de Colores**: 
   - **Abismo**: `#020617` (Fondo principal)
   - **Oro Champagne**: `#d4af37` (Acentos y tipografía elegante)
   - **Bordes**: Gradientes sutiles entre `#1e293b` y `#334155`.

## Regla de Oro
EL CONTENIDO ES SAGRADO. Los datos (`DATA` array), links de Cloudinary y nombres de comisiones no se modifican bajo ninguna circunstancia durante el refinamiento estético.
