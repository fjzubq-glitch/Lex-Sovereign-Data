# Skill: Lex SEO Viral Engine

Define la estrategia de visibilidad y crecimiento orgánico de LexSearch Gold.

## Estrategia de Metadatos
1.  **Páginas Dinámicas**: Cada ley debe generar un `title` y `meta-description` únicos que incluyan el número y título oficial.
2.  **OpenGraph**: Configurar `og:title` y `og:image` dinámicos para que, al compartir un enlace en redes (X, LinkedIn, WhatsApp), la previsualización sea profesional y atractiva.
3.  **Keywords**: Enfoque quirúrgico en términos como "derecho argentino", "jurisprudencia", y nombres de fallos específicos (ej. "Halabi").

## Configuración Técnica
- Implementación de React Helmet o similar para SSR/Header injection.
