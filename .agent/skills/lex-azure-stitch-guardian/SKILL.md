# Skill: Lex Azure Stitch Guardian

Garantiza que la identidad visual de LexSearch Gold se mantenga inalterable bajo el protocolo Azure Stitch.

## Especificaciones de Diseño
1.  **Wallpaper**: `#F0F7FF` (Azul Azure) debe ser el fondo global.
2.  **Tipografía**: `#050505` (Negro Obsidian) para todo el texto de lectura.
3.  **Bordes**: `0.5px` (quirúrgicos) en todos los contenedores y `index.css`.
4.  **Botones**: Estilo square (0px radius) para una estética de precisión institucional.

## Verificación de Estilo
- Ningún cambio de CSS debe alterar el `Azure Background`.
- Se prohíben bordes de 1px o superiores.
